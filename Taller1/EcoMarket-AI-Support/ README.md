# EcoMarket AI Support

Solución propuesta para el Taller Práctico #1 sobre optimización de la atención al cliente mediante IA generativa.

## Objetivo
Diseñar una solución de IA para responder consultas repetitivas de clientes en EcoMarket, reduciendo tiempos de respuesta y manteniendo calidad y control sobre la información.

## Estructura del proyecto
- `docs/fase1_modelo.md`: selección y justificación del modelo
- `docs/fase2_riesgos.md`: fortalezas, limitaciones y riesgos éticos
- `data/orders.json`: base de datos simulada de pedidos
- `data/return_policies.json`: reglas simuladas de devolución
- `src/main.py`: script de prueba
- `src/prompts.py`: prompts diseñados

## Ejecución
```bash
python src/main.py
````
